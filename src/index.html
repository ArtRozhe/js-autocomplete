Example 3

The setting generateLayoutSuggestion callback:

          let localDataProviderEx3 = new LocalDataProvider({
            data: [
              'Abasto',
              'Acassuso',
              'Acebal',
              'Acevedo',
              'Adelia Maria',
              'Agua de Oro',
              'Albardon',
              'Albarellos',
              'Alberdi',
              ...
            ]
          });

          let autoCompleteEx3 = new AutoComplete({
            containers: '.auto-complete_ex-3',
            dataProvider: localDataProviderEx3,
            minChars: 1,
            delay: 50,
            generateLayoutSuggestion: function(suggestionText, searchingText) {
              let resultHtml = '';
              const re = new RegExp(`(${searchingText.split(' ').join('|')})`, 'gi');

              resultHtml = `${resultHtml}<div>${suggestionText.replace(re,
                '<span style="color: #FF5722;">$1</span>')}</div>`;

              return resultHtml;
            }
          });

Example 4

For this example you need to start the server with command "npm run server". The server will start at http://localhost:3003

  • data provider: ApiDataProvider
  • AutoComplete options:

          let autoCompleteEx4 = new AutoComplete({

            delay: 50
          });

Options

AutoComplete component

  • autoRun - auto run of the component (default: true);
  • containers - DOM containers (array, selector, or single DOM container) (default: null);
  • delay - the number of milliseconds that the component waits to send a request to the data provider (default: 100);
  • minChars - the minimum number of entered characters to send requests to the data provider (default: 3);
  • generateLayoutSuggestion - the markup generation function for each suggestion (default: function);
  • dataProvider - the data provider to which the component sends a request to receive the data set (default: null);

LocalDataProvider

  • data - local data for filtering (default: null);
  • selectionCb - search strategy in an array of data for a given phrase (default: function);
  • useCache - use or not use the cache (default: true);

ApiDataProvider

  • apiPath - path to the api (default: null);
  • useCache - use or not use the cache (default: true);
