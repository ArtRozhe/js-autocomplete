To see the demos clone this repo and open index.html (dist/index.html) in the browser.
> :warning: For the example with the ApiDataProvider you need to start the server. See below.

### Install dependencies
```bash
npm install
```

### Development mode
```bash
npm run dev
```

### Production mode
```bash
npm run build:prod
```

### Starting server
```bash
npm run server
```
